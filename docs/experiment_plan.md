# 实验计划

## 目标

为数字经济学报告生成可量化的实验数据，支持以下论点：

1. AIGC 3D 资产能显著降低商品信息不对称（从 2D 图像到可旋转 3D）。
2. 模型压缩可以让网页端加载成本接近现有图片量级。
3. 二维虚拟试穿降低退换货带来的搜索/物流成本。

## 数据存档

每次实验任务都会在 `outputs/jobs/<job_id>/` 下生成：
- `job.json` 状态
- `job.log` 日志
- `metrics.json` 指标
- `exports/` 资产文件

把这些目录打包到 `outputs/reports/exp_<YYYYMMDD>.zip` 作为报告附件。
